# Генератор Задач (Task Generator)

**Автор:** [Ваше Имя и Фамилия]

## Описание
Приложение на Python (tkinter) для генерации случайных задач из предопределенного списка. Позволяет отслеживать историю сгенерированных задач, фильтровать их по типу (учеба, спорт и т.д.) и сохраняет историю в JSON-файл между запусками.

## Функционал
1.  **Генерация:** Кнопка «Сгенерировать задачу» выбирает случайную задачу из списка.
2.  **Типизация:** Задачи разделены на категории: Учеба, Спорт, Работа, Здоровье.
3.  **История:** Отображение списка выполненных задач (последние сверху).
4.  **Фильтрация:** Возможность отфильтровать историю по типу задачи.
5.  **Сохранение:** История автоматически сохраняется в `tasks.json`.

## Примеры использования
1.  Запустите скрипт: `python app.py`.
2.  Нажмите кнопку "Сгенерировать задачу".
3.  Посмотрите на появившуюся задачу в метке и в списке истории.
4.  Выберите тип "Спорт" в выпадающем списке, чтобы увидеть только спортивные задачи.
